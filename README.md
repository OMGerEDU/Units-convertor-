### Overview
This repository contains **single-page landing templates** for various professions, fully client-side and ready to deploy on Vercel. Each template is built with **Next.js**, **React**, and **Tailwind CSS**, and includes customizable color palettes, lightweight widgets, and no backend dependencies.

### Features
- 🏠 **Single Page** – All content on one scrollable page
- 🎨 **Customizable Themes** – Predefined color palettes for each profession
- ⚙️ **Widgets & Plugins** – Pure CSS/JS widgets (accordions, sliders, lightboxes, maps)
- 🚀 **Vercel Ready** – Zero-config deployment with `vercel` CLI
- 📱 **Responsive** – Mobile-first design, works on all devices

### Tech Stack
- **Framework**: Next.js (React)
- **Styling**: Tailwind CSS
- **Bundler**: Vercel (serverless deployment)
- **Plugins**: Swiper.js, LightGallery, ScrollReveal.js, Pure CSS components

### Quick Start
1. **Clone the repo**
   ```bash
   git clone https://github.com/omerdahan/landing-templates.git
   cd landing-templates
   ```
2. **Install dependencies**
   ```bash
   npm install
   ```
3. **Run locally**
   ```bash
   npm run dev
   # Open http://localhost:3000
   ```
4. **Deploy to Vercel**
   ```bash
   npm install -g vercel
   vercel login
   vercel --prod
   ```

### Folder Structure
```
/pages           # Next.js pages for each template
/components      # Reusable UI components and widgets
/styles          # Tailwind configuration and theme files
/public          # Static assets (images, videos, icons)
```
